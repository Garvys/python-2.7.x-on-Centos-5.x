# DESCRIPTION:

Upgrade python (http://www.python.org/) to latest stable version (as of 04/02/12) without breaking yum on CentOS 5.x.

# REQUIREMENTS:

This script has only been tested on CentOS 5.6, 5.7 and 5.8.

# USAGE:

You must be root or use sudo.

	#> ./install_python27.sh
